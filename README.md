# codex-web

一个零依赖的极简短信发送平台 Demo，可直接用浏览器打开 `index.html`，或通过任意静态文件服务器运行。

## 功能

- 填写收件手机号和 1-160 字短信内容
- 前端校验手机号与空内容
- 模拟短信发送成功提示
- 使用 `localStorage` 保存最近 10 条发送记录
- 支持一键清空发送记录

## 本地运行

```bash
python3 -m http.server 4173
```

然后访问 <http://127.0.0.1:4173/>。

> 当前版本是前端演示版，不会真正调用短信服务商 API。接入真实短信服务时，请在后端保存密钥并调用 Twilio、Vonage、阿里云短信等服务，避免在浏览器端暴露密钥。
